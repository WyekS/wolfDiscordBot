Puedes usar los siguientes comandos (para los alias no usar los <>):
1. /wolf help
/* Muestra la ayuda del bot *
2. /wolf start <alias>
/* Primero para el servidor activo y luego inicia el servidor indicado *
3. /wolf stop <alias>
/* Parar el servidor indicado con el alias *
4. /wolf restart <alias>
/* Reinicia el servidor indicado con el alias *
5. /wolf status
/* Muestra el estado de todos los servidores *
6. /wolf update all|servers|mods
/* Elegir entre las opciones marcadas *
7. /wolf slot X <especialización>
/* Crea un evento con X nuevas vacantes de especialización *
