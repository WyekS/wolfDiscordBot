# ===================
# Ayuda Wolf Team Bot
# ===================

Puedes usar los siguientes comandos , para los alias no usar <>:
1. /wolf help     /* Muestra la ayuda del bot *
2. /wolf start <alias>     /* Inicia el servidor indicado con el alias *
3. /wolf stop <alias>     /* Parar el servidor indicado con el alias *
4. /wolf restart <alias>     /* Reinicia el servidor indicado con el alias *
5. /wolf nuclear     /* Para y reinicia todos los servidores de WolfTeam *
6. /wolf status     /* Muestra el estado de todos los servidores *
7. /wolf update all|servers|mods    /* Elegir entre las opciones marcadas *
8. /wolf slot X <especialización>     /* Crea un evento con X nuevas vacantes de especialización *
